# Contributing

Thank you for your interest in contributing to the STATE16 Physical AI Community.

You can contribute in several ways:

- Join discussions
- Review papers
- Suggest failure modes
- Propose benchmark ideas
- Write documentation
- Create toy examples
- Share notebooks
- Help improve the glossary

## Contribution flow

1. Start with a GitHub Discussion if the idea is broad.
2. Open an Issue if the work is clearly scoped.
3. Submit a Pull Request if you want to add or edit content.
4. A maintainer will review the contribution before it is merged.

## Scope

This repository is for open community discussion, research exploration, education, and shared technical work around Physical AI Assurance.

Please do not contribute:

- Confidential data
- Customer information
- Private algorithms
- Proprietary commercial logic
- Internal product code
- Third-party confidential material
